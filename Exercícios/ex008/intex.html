Principais formatações

Negrito / Desatque

Nessa frase temos um termo em negrito usando a tag 'b' (mas não é semântica)

Nesta frase temos um termo em destaque usando a tag 'strong' (semântica)

Itálico / Ênfase

Nest frase temos um termo em itálico usando a tag 'i' (não semântica)

Nesta frase temos um termo em ênfase usando a tag 'em' (sêmantica)

Texto Marcado

Podemos criar também um texto marcado usando tag 'mark' (semântica)

E em outro parágrafo temos outro texto marcado no final

Texto grande e pequeno

Estamos criando um Texto Grande e um Texto Pequeno

Texto Deletado / Excluído

Podemos marcar um texto como excluído pra indicar que ele deve ser lido,mas não considerado usando a tag 'del'

Texto Incuído

Podemos marcar um texto inserido para dar ênfase e indicar que ele foi adicionado depois,usando a tag 'ins'

Texto sobrescrito

Para inserir coisas do tipo x20+3 usando a tag 'sup'

Texto subescrito

Para inserir textos do tipo H2O usando a tag 'sub'
